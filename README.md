# Grokking Deep Learning

Deep learning implementations from scratch using only NumPy. Based on the book "Grokking Deep Learning".

## Chapters

- **Chapter 3**: Neural network basics (forward propagation)
- **Chapter 4**: Gradient descent and learning
- **Chapter 5**: Batch training and vectorization
- **Chapter 6**: Backpropagation
- **Chapter 7**: Activation functions (ReLU, sigmoid, tanh)
- **Chapter 8**: Regularization (dropout, early stopping)
- **Chapter 9**: Softmax and cross-entropy
- **Chapter 10**: Convolutional Neural Networks (CNNs)
- **Chapter 11**: Recurrent Neural Networks (RNNs)
- **Chapter 12**: Long Short-Term Memory (LSTMs)
- **Chapter 13**: Word embeddings (Word2Vec)

## Requirements

- Python 3.x
- NumPy
